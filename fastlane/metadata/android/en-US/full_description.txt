<i>TouchDroid</i> is an Android Application to Emulate inpute devices i.e. Touchpad and Keyborad to control Windows computer via local Network. <a href='https://www.github.com/Akshayaap/TouchServer' target='_blank' rel='nofollow'>TouchServer</a> is reuired to be Running on Windows Operating System in order to capture data sent by Application from network and to register Appropreate events to Host Operating System.
